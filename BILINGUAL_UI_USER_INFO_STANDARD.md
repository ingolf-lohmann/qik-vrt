# BILINGUAL_UI_USER_INFO_STANDARD

## Deutsch

Alle interaktiven Benutzeroberflächen und alle nutzergerichteten Bundle-Informationen müssen zweisprachig Deutsch/Englisch sein.

Dies gilt insbesondere für:

```text
Startmeldungen
Blockmeldungen
Done-/Pass-Meldungen
Lizenz-/Urheberrechtshinweise
Akzeptanzabfragen
Gegenstellentyp-Abfragen
OTHER-Beschreibungsabfragen
README
NOTICE
QALL.ini-Kommentare
Nutzungshinweise
```

## English

All interactive user interfaces and all user-facing bundle information must be bilingual German/English.

This applies especially to:

```text
startup messages
block messages
done/pass messages
license/copyright notices
acceptance prompts
counterparty type prompts
OTHER description prompts
README
NOTICE
QALL.ini comments
usage instructions
```

## Blocker

```text
USER_FACING_TEXT_NOT_BILINGUAL_DE_EN
```
