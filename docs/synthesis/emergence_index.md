## Emergenzregel

Dieses Repository emergiert nicht dadurch, dass ein einzelner Artikel als Dogma gesetzt wird. Es emergiert, indem wiederkehrende Unterschiede aus den Artikeln extrahiert und in eine höhere Anschlussordnung gebracht werden.

## Wiederkehrende Kerndifferenzen

1. **Fakt / Deutung / Hypothese / Rauschen / Manipulation**
2. **lokale Erfahrung / globale Kausalstruktur**
3. **Information / Träger**
4. **Prognose / Ereignis**
5. **Wiederholung / Wahrheit**
6. **Sichtbarkeit / Nachweis**
7. **Macht / Verantwortung**
8. **technische Fähigkeit / ethische Freigabe**
9. **Kausalität / praktische Retrokausalität**
10. **ungeprüfte Autorität / auditierbare Ordnung**

## Gesamtformel

```text
Unterschied -> Bestimmbarkeit -> Information -> Kausalität -> Anschluss -> Wirkung -> Nachweis -> Verantwortung -> Ethik
```

## Emergenzschluss

Die neuen Artikel bilden zusammen keine bloße Meinungssammlung. Sie bilden eine operationalisierbare Wirkraumordnung: Information wirkt über Zeit, Ort, Träger, Systeme und Menschen hinweg. Deshalb muss sie unterscheidbar, prüfbar, versionierbar, auditierbar und ethisch gebunden werden.
