# Competition eligibility checklist

Complete this checklist against the official rules before any submission.

## Project facts already prepared

- [x] Public GitHub repository
- [x] Clear technical README
- [x] Reproducible local test command
- [x] Fixed release tag and full commit SHA
- [x] Content-tree and document hashes
- [x] Successful hosted CI and Pages runs
- [x] Source, documentation, historical, and third-party license boundaries
- [x] Short abstract, long pitch, evidence matrix, and demo script
- [x] Community, security, support, and governance drafts
- [ ] Independent reproduction by an unaffiliated third party
- [ ] External security assessment
- [ ] DOI for this exact release
- [ ] Final demo video or GIF

## Organizer-specific checks

- [ ] Official competition name and rules URL
- [ ] Submission URL and authenticated participant account
- [ ] Deadline, time zone, and permitted updates after submission
- [ ] Category and scoring weights
- [ ] Self-nomination, age, residence, team-size, and entity restrictions
- [ ] Existing-project and development-window requirements
- [ ] Required repository visibility and license type
- [ ] Whether PolyForm Noncommercial 1.0.0 is eligible
- [ ] Required language and word/character limits
- [ ] Required presentation, video, screenshots, or live demo
- [ ] Disclosure of AI-assisted work
- [ ] Rights granted to the organizer over submission materials
- [ ] Identity, tax, travel, and prize-acceptance requirements

## License stop condition

If the rules require an OSI-approved open-source license, the active release is
not currently eligible: PolyForm Noncommercial 1.0.0 is source-available and
restricts commercial use. Do not relabel it as OSI open source or change the
license solely as a marketing step. Any relicensing decision requires a
separate rights and strategy decision by the rights holder.

## Submission stop condition

Do not submit to a competition that cannot be identified on an official,
current organizer page. GitHub's **For the Love of Code 2025** ended on
2025-09-22; its page is evidence of a closed event, not a 2026 call.
