QIK-VRT Effect Gate is an auditable release layer for multi-agent systems. It separates technical success from authorized real-world effect: TRANSPORT_ACK != EFFECT_ACK. Three roles collect evidence, assess risk and policy, and verify release. A bounded gate returns NACK, CONTINUE, DONE, ISOLATE, or BLOCK; only DONE permits release. The public reference passes 102 tests. The GOAI extension maps it to AgentTeams with reusable Skills, shared state, traces, human approval, rollback, and audit.
