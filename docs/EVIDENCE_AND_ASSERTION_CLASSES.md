# Evidence and Assertion Classes

## Assertion classes
FACT, OBSERVATION, DOCUMENTATION, TESTIMONY, HYPOTHESIS, SUSPICION, RISK, INTERPRETATION, SPECULATION, REFUTED.

## Promotion rule
An assertion may be promoted only when additional traceable evidence exists and counter-evidence was checked.

## Demotion rule
An assertion must be demoted if evidence is weakened, contradicted, or shown to be improperly sourced.

## Consequence rule
Consequences must match assertion class and risk level.
A hypothesis may justify investigation.
A plausible concrete risk may justify protection.
A fact may justify stronger decision, subject to proportionality and due process.

## Final-pass rule
No final-pass is valid if assertion class is missing.
No final-pass is valid if counter-evidence was not considered.
