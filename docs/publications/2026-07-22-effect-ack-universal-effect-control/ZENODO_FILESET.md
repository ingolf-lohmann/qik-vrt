- `ZENODO_SHA256SUMS`

`ZENODO_SHA256SUMS` binds the exact deposited files except itself. Zenodo's
server-reported file list and MD5 values remain separate transport evidence;
they do not replace this SHA-256 manifest.

The executable `effect_ack_universality_proof.py` is PolyForm Noncommercial
source code and is deliberately not relicensed or uploaded as a Working Paper
file. The Internet-Draft XML and TXT retain their IETF Trust notices and are
also excluded from this record. Both remain part of the linked, versioned
QIK-VRT software snapshot together with the complete repository bundle and its
repository-wide `SHA256SUMS`.

This separation preserves the reproducibility boundary without applying one
Zenodo deposition license to files that carry different legal notices.
