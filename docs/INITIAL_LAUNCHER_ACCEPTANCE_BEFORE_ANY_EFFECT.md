# Initial Launcher Acceptance Before Any Effect

V29 repariert den reflexiven Acceptance-Fehler im Launcher.

```text
Akzeptanz vor Wirkung.
```

Ohne persistierten Acceptance-Record darf der Launcher keinen `command_start` erzeugen.
