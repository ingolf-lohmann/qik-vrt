# Windows Keep-Open Launcher

The Windows acceptance launcher must not disappear immediately after an elevated or double-click start.

Rules:
- QIKVRT.cmd acceptance writes console output to `qikvrt/runtime/win_acceptance/console.out.txt`.
- stderr is written to `console.err.txt`.
- the exit code is printed as `WINDOWS_ACCEPTANCE_CMD_EXIT`.
- the console stays open with `pause` unless `QIKVRT_NO_PAUSE=1` is set.
- a final visible pointer to WIN_ACCEPTANCE_RESULT.tsv/json is printed.

This fixes the class `WINDOWS_ELEVATED_POWERSHELL_DISAPPEARS_WITHOUT_VISIBLE_RESULT`.
