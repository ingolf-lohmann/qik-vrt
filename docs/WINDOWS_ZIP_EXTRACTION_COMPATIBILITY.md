# Windows ZIP Extraction Compatibility

V36 repariert den Delivery-Fehler: Das ZIP muss unter Windows entpackbar sein.

Der entscheidende Unterschied zu vorherigen Lieferungen ist der kurze technische Paketname:

```text
QV36_WINZIP_OK.zip
QV36/
```
