# QIKVRT Bilingual Documentation / Zweisprachige Dokumentation

## Deutsch
Alle Dokumentations- und sonstigen Nicht-Source-Code-Artefakte müssen zweisprachig anschlussfähig sein. Das bedeutet mindestens: ein sichtbarer deutscher und englischer Dokumentationskopf, ein sichtbarer deutscher und englischer Dokumentationsfuß, eindeutige Urheber-/Rechte-/Lizenzangaben und maschinenprüfbare Gates.

## English
All documentation and other non-source-code artifacts must remain bilingual-accessible. This means at minimum: a visible German and English document header, a visible German and English document footer, explicit authorship/rights/license information, and machine-checkable gates.

## Pflicht / Requirement
- Deutsch und Englisch sichtbar / German and English visible
- Urheber sichtbar / author visible
- Lizenz sichtbar / license visible
- Kein Final-Pass ohne zweisprachige Dokumentation / no final pass without bilingual documentation
- Source-Code bleibt technisch klar; Kommentare und Dokumente bleiben DE/EN anschlussfähig / source code remains technically clear; comments and docs remain DE/EN accessible
