# Sicherheits- und Grenzregeln

Dieses Repository ist ein defensives Nachvollziehbarkeitsartefakt.

Es ist ausdrücklich nicht:

- ein Überwachungswerkzeug,
- eine Beschuldigungsmaschine,
- ein amtlicher Rechtsakt,
- ein Ersatz für Polizei, Justiz, Medizin oder anwaltliche Beratung,
- ein Mittel zur Veröffentlichung sensibler personenbezogener Daten.

Sicherheitsgrundsatz: Prüfbarkeit darf nicht dazu führen, Menschen unnötig zu gefährden oder sensible Details missbrauchbar zu veröffentlichen.

Datenschutzgrundsatz: So viel Nachvollziehbarkeit wie nötig, so viel Privatheit wie möglich.


## V2.2 explizite Boundary-Sätze

Kein amtlicher Rechtsakt.
Kein Überwachungsinstrument.
Kein Instrument zur Beschuldigung konkreter Dritter.
No final pass without traceability.
No final pass unless multicast and ontology selftests pass.

## V2.3 boundary extension
The governance implementation is procedural and defensive. It must not be used as a surveillance instrument, public humiliation system, or substitute for lawful due process.
