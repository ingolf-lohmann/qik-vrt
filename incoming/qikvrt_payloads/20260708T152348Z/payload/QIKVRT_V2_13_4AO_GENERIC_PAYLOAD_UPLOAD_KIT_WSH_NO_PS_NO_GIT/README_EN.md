# QIKVRT V2.13.4AO Generic Payload GitHub Upload Kit

This kit uploads arbitrary payload files into an existing QIK-VRT GitHub node.

Windows path:

1. Put files or folders into `payload\`.
2. Double-click `START_HIER_QIKVRT_UPLOAD.cmd`.
3. Accept copyright/license with `JA` or `YES`.
4. Authorize Product Owner upload with `JA` or `YES`.
5. Run `DRYRUN` first.
6. Then choose `UPLOAD` and paste a GitHub fine-grained token.

The Windows primary path does not execute `.ps1` files and does not use Git.
