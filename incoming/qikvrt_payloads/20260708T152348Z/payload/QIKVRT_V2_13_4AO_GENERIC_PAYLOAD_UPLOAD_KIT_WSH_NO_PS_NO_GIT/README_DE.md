# QIKVRT V2.13.4AO Generic Payload GitHub Upload Kit

Dieses Paket persistiert beliebige Dateien aus `payload/` in einen bestehenden QIK-VRT GitHub-Knoten.

Windows-Weg:

1. Dateien oder Ordner nach `payload\` legen.
2. `START_HIER_QIKVRT_UPLOAD.cmd` doppelklicken.
3. Urheberrecht/Lizenz mit `JA` oder `YES` akzeptieren.
4. Product-Owner-Upload mit `JA` oder `YES` autorisieren.
5. Zuerst `DRYRUN` ausfuehren.
6. Danach `UPLOAD` waehlen und GitHub Fine-Grained Token einfuegen.

Der Windows-Primärpfad verwendet keine PowerShell-Datei und kein Git.
