# Angebot und Preismodell

## 1. Repository Trust Audit

Ziel: Ein Repository prüfen, Nachweislücken identifizieren, QIK-VRT-kompatiblen Verbesserungsplan liefern.

Leistung:
- Root-Strukturprüfung
- Lizenz-/Urheberstatus-Prüfung
- Release-/Hash-/Evidence-Prüfung
- CI/CD-/Workflow-Sichtprüfung
- AI-Act-/NIS2-/CRA-Nachweislandkarte
- Executive Summary

Startpreis: 750 bis 2.500 EUR je Repository.

## 2. Seed/Node Setup

Ziel: Ein Seed/Node-Mesh für ein Team aufsetzen.

Leistung:
- Seed-Repository konfigurieren
- Node-Repository anbinden
- Heartbeat, Revalidation, Audit, Dashboard
- Product-Owner-Schulung

Startpreis: 2.500 bis 7.500 EUR.

## 3. Compliance Evidence Pack

Ziel: Nachweisakte für Audit, Vorstand, Datenschutz, Rechtsabteilung oder Kundenprüfung.

Leistung:
- Auditbericht
- Evidence-Index
- Release-Freeze
- Risiko-/Boundary-Dokument
- technische Dokumentationsmatrix

Startpreis: 1.500 bis 5.000 EUR.

## 4. Managed Seed

Ziel: Fortlaufender Betrieb eines Seed-Registry-Knotens.

Leistung:
- regelmäßige Revalidation
- Dashboard
- Statusberichte
- Incident-/Revocation-Pfad

Preis: 99 bis 999 EUR monatlich, abhängig von Node-Zahl und Support-Level.

## 5. Enterprise / Behörde

Ziel: QIK-VRT als interne Trust-Mesh-Schicht.

Leistung:
- private Seed-Registry
- Rollen-/Rechtekonzept
- Onboarding-Prozesse
- Dokumentationsmapping
- Audit-Exports
- Schulung

Preis: individuell.

## Kommerzielle Grenze

Diese Preise sind Startanker für Markttests, nicht endgültige Listenpreise. Rechtsberatung, Zertifizierung durch Benannte Stellen und externe Sicherheitsprüfung werden separat beauftragt.
