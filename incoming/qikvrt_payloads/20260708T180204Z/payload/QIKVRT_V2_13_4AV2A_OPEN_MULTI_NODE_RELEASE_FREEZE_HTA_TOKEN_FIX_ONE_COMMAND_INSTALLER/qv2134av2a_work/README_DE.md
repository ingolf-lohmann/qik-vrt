# QIKVRT V2.13.4AV2A Open Multi-Node Release Freeze HTA Token Fix

Start unter Windows:

```cmd
START_HIER_RELEASE_FREEZE_INSTALL.cmd
```

Dieses Paket ersetzt 4AV2, falls der maskierte HTA-Token-Dialog mit Syntaxfehler abbricht.

Eigenschaften: ein Command, maskierter lokaler Token-Dialog, kein PowerShell-Ausführungspfad, keine `.ps1`, kein Git-Kommando, kein eingebetteter Token.
