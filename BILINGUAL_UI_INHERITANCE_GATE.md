# BILINGUAL_UI_INHERITANCE_GATE

## Deutsch

Die zweisprachige UI- und Nutzerinformationspflicht Deutsch/Englisch muss von allem geerbt werden, was aus diesem Bundle, Repository oder GitHub heraus erzeugt wird.

Das gilt für:

```text
Clone
Download
Fork
Template-Instanz
Merge
Release
Repackage
Build-Artefakt
lokale Reproduktion
GitHub-Persistenz-Ausgabe
Repository-Payload
Folge-Bundle
```

## English

The bilingual German/English UI and user-information requirement must be inherited by everything derived from this bundle, repository, or GitHub.

This applies to:

```text
clone
download
fork
template instance
merge
release
repackage
build artifact
local reproduction
GitHub persistence output
repository payload
follow-up bundle
```

## Blocker

```text
BILINGUAL_UI_INHERITANCE_MISSING
DERIVED_ARTIFACT_USER_INFO_NOT_BILINGUAL_DE_EN
CLONE_OR_MERGE_DROPPED_BILINGUAL_UI_STANDARD
```
