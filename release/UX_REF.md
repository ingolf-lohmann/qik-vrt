# QIKVRT V2.13.4 UX / License / Deploy Reference

- Public Windows entry point remains QIKVRT.cmd.
- Public POSIX entry point remains QIKVRT.sh.
- License/rights/authorship acceptance is mandatory before any non-help activity.
- GitHub deployment prompts for GITHUB_TOKEN at runtime when needed and never persists it.
- Windows entry point pauses by default unless QIKVRT_NO_PAUSE=1.
- Windows acceptance prefers zigcc before a known incomplete standalone LLVM clang path.
