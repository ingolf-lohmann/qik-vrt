# SCRIPT_ONLY_FALSE_DELIVERY_ACCEPTANCE_TEST

## Fehlerklasse

`SCRIPT_ONLY_PACKAGE_DELIVERED_WHEN_CONTENT_PACKAGE_REQUIRED`

Deutsch:

`NUR_SKRIPTPAKET_GELIEFERT_OBWOHL_CONTENTPAKET_GEFORDERT`

## Regel

Wenn der Nutzer tatsächlichen Repository-/Monatscontent verlangt, darf kein bloßes Script-only-Paket ausgeliefert werden.
