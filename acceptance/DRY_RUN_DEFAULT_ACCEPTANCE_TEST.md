# DRY_RUN_DEFAULT_ACCEPTANCE_TEST

## Fehlerklasse

`DESTRUCTIVE_OPERATION_NOT_DRY_RUN_BY_DEFAULT`

Deutsch:

`ZUSTANDSVERAENDERUNG_NICHT_STANDARDMAESSIG_DRY_RUN`

## Regel

Zustandsverändernde externe Operationen müssen standardmäßig dry-run sein und explizit aktiviert werden.
