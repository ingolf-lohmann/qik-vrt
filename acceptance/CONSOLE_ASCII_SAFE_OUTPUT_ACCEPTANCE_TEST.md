# CONSOLE_ASCII_SAFE_OUTPUT_ACCEPTANCE_TEST

```text
CONSOLE_OUTPUT_ENCODING_UNSAFE_FOR_WINDOWS_CODEPAGE
SCRIPT_STDOUT_STDERR_CONTAINS_UMLAUTS
```

CMD/BAT/PowerShell/Shell launcher output must use ASCII-safe wording. German umlauts are allowed in higher-level documents and UIs, not in raw console launcher text.
