# GITHUB_RUNTIME_PERSISTENCE_ACCEPTANCE_TEST

## Fehlerklasse

`GITHUB_RUNTIME_PERSISTENCE_WITHOUT_EVIDENCE`

Deutsch:

`GITHUB_LAUFZEITPERSISTENZ_OHNE_EVIDENZ`

## Regel

GitHub-Persistenz muss Remote, Branch, Commit-Hash, Push-Status und Workflow-/CLI-Evidence dokumentieren.
